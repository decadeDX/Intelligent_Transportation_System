车速实时检测测试（/speedVideoDetected）

/

请通过 http://127.0.0.1:8000/speed_detected.html 打开本页（不要双击本地 html 文件）。
选择视频后点「开始」，消费 POST /speedVideoDetected 的 NDJSON 流。测速请保持 frame_interval=1。
仅统计车辆；测速带内用接地点 x/y 二维回归求速（横向车流不再只算垂直位移）。
默认 frame_interval=1、米/像素=0.1；标定请在蓝线高度量车道宽（如 3.5m÷35px=0.1）。
标定示例：车道宽约 3.5m、在画面中约 70 像素 → 填 3.5 / 70，或直接把米/像素设为 0.05。
任务结束后展示速度汇总，完整 API 响应保存在 upload/detected/{uuid}/result.json。
